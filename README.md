# Double-Life
 
Double Life is a life simulation videogame where the player can create characters that resemble real-life people.
These characters, or Doubles, live in a virtual city where they will start friendly relationships with other Doubles
while the player takes care of them.

Created a system that gives Doubles needs like hunger, loneliness, etc.
Created a system that allows the player to decorate a Double's room using furniture.
Created a system that allows the player to manage their Double population. This includes adding, editing and
deleting a Double.
Added a world hub where the player can interact with buildings and do various activities. Added a
system where the Doubles can communicate with the player according to their needs. Added a
relationship system where Doubles can be friends, lovers or enemies with other Doubles. Added a
character creator where the player can create or edit Doubles.
